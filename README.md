#  Marketing Campaign Analysis using Python

## Project Overview

This project analyzes marketing campaign data to understand customer behavior, spending patterns, and factors influencing campaign acceptance. The objective is to generate actionable business insights that can help improve customer targeting and marketing effectiveness.

## Business Problem

Businesses invest heavily in marketing campaigns, but not all campaigns achieve the desired response. This project explores customer demographics, purchasing behavior, and campaign outcomes to identify the characteristics of customers who are more likely to respond positively to marketing efforts.

## Project Workflow

### 1. Data Cleaning

* Handled missing values
* Corrected data types
* Standardized column names

### 2. Feature Engineering

* Created Age variable
* Calculated Total Spending
* Derived Children Count
* Calculated Total Purchases

### 3. Exploratory Data Analysis (EDA)

* Customer demographic analysis
* Spending pattern analysis
* Campaign response analysis
* Correlation analysis

### 4. Statistical Analysis

* Performed hypothesis testing
* Compared customer groups based on campaign acceptance
* Evaluated statistical significance using t-tests

### 5. Data Visualization

* Correlation Heatmap
* Income vs Spending Analysis
* Campaign Response Analysis
* Product Category Performance

## Key Insights

* Higher-income customers tend to spend more and respond better to marketing campaigns.
* Spending behavior varies across different demographic groups.
* Customers with fewer dependents generally exhibit higher spending patterns.
* Certain product categories contribute significantly to overall revenue.
* Targeted marketing strategies can improve campaign effectiveness and customer engagement.

## Project Outcomes

The analysis provides valuable insights into customer behavior and marketing performance, enabling businesses to make data-driven decisions for better campaign planning and customer segmentation.
